# 🌍 AI Trip Planner

AI Trip Planner is an intelligent web application that helps users plan their trips with the assistance of the **Microsoft Phi-3 Mini 4K Instruct AI model**. Users can simply enter their travel requirements, and the chatbot generates personalized travel plans, including itineraries, budget estimation, hotel suggestions, tourist attractions, local food recommendations, weather information, and packing lists.

## ✨ Features

- 🤖 AI-powered travel chatbot
- 🗺️ Personalized trip planning
- 📅 Day-wise itinerary generation
- 💰 Budget estimation
- 🏨 Hotel recommendations
- 📍 Tourist attractions
- 🍴 Local food suggestions
- 🌦️ Weather information
- 🎒 Packing checklist
- 🚕 Local transportation suggestions
- 📱 Responsive and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Microsoft Phi-3 Mini 4K Instruct
- Hugging Face Inference API
- Vercel Edge Functions

## 🚀 How It Works

1. Enter your travel query.
2. The request is sent to the backend.
3. The backend communicates with the Microsoft Phi-3 AI model.
4. The AI generates a personalized travel plan.
5. The response is displayed in the chatbot interface.
